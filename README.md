## 项目介绍

用户和电脑轮流在九宫格画圈或者叉，谁方先连成横、竖、斜线即获胜。

## 项目使用技术

1. Grid 布局实现九宫格；
2. minmax 算法实现计算机寻求步数最优解。
3. CSS variable 增加代码的可复用性。
